This cipher doesn't have a real name

This is a cipher that I thought of myself, I got the idea from the image displayed at the bottom right of this page. This is a simple substitution cipher where every letter in the English alphabet can be substituted with one of 4 double digit numbers, apart from the letters 'wxyz' which can only be one of 3 double digit numbers. Encryption is very simple, for every letter inputted, choose one of the possible outputs. To decipher the encryption all you need to know is which numbers correspond to which letters in the alphabet.
To program this cipher I came up with an algorithm which uses a random number generator and simple maths to encrypt a message, and straight forward maths to decrypt a given encrypted message back to English. Click this link for more information about number ciphers
